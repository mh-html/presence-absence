پروژه حضورغیاب تحت وب
1. ابتدا گزینه "افزودن دانش آموز" را بزنید
2. اطلاعات مربوطه را پر کنید
3. برای ایجاد کلاس گزینه "تاریخ های برگزاری کلاس" را بزنید
4. یک تاریخ جدید ثبت کنید
5. روی تاریخ مورد نظر کلیک کنید
6. دانش آموزانی که حضور ندارند را غیبت بزنید
7. از قسمت "لیست دانش آموزان" تمام اطلاعات را مشاهده بفرمایید
